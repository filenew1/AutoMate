# 异常证照

该技能用于登录深圳市政务系统，查找状态异常的证照信息。

## 功能说明

1. 打开深圳市政务系统登录页面
2. 检查是否已登录，如已登录则跳过登录步骤
3. 输入用户名"admin"
4. 等待用户输入密码和验证码，点击登录后进入首页，并存储cookie
5. 点击"证照管理"菜单，再点击"证照签发"菜单
6. 获取表格内容，筛选出状态异常的证照（状态列为红色）
7. 显示异常证照的批复文号和状态两列信息

## 使用场景

- 用户需要查询系统中状态异常的证照
- 用于检查证照签发状态，发现问题证照
