# 修复欢迎页布局以匹配原型设计

## 问题分析

当前 WelcomePage.tsx 的实现与原型设计（MainLayout.html）存在以下差异：

### 1. 标题和描述内容不匹配
- **原型**：标题"欢迎使用 AutoMate"，描述"选择左侧的智能体开始对话，体验智能化的任务处理"
- **当前**：标题"欢迎使用 AutoMate"，描述"您的智能体交互平台"

### 2. 统计卡片样式不匹配
- **原型**：
  - 卡片背景：`bg-gradient-to-br from-gray-100 to-gray-200`
  - 统计数字：`text-blue-600`、`text-green-600`、`text-purple-600`
  - 统计标签：`text-xs text-gray-600 mt-1`
- **当前**：
  - 卡片背景：`bg-white dark:bg-gray-800`
  - 统计数字：`bg-gradient-to-r from-blue-500 to-purple-600 bg-clip-text text-transparent`
  - 统计标签：`text-sm text-gray-600 dark:text-gray-400`

### 3. 快速开始引导结构不匹配
- **原型**：
  - 3个步骤，使用 `quick-start-guide` 和 `quick-start-item` 类
  - 步骤内容：
    1. 从左侧列表选择一个智能体
    2. 在输入框中输入您的问题
    3. 按 Enter 发送消息开始对话
- **当前**：
  - 4个步骤，使用 `bg-gray-50 dark:bg-gray-700 rounded-lg` 类
  - 步骤内容：
    1. 添加智能体 - 创建和管理您的智能体，配置技能和参数
    2. 开始对话 - 选择智能体并开始对话，体验智能交互
    3. 使用技能 - 为智能体配置技能，扩展智能体能力
    4. 管理文件 - 上传和管理文件，支持文件交互

### 4. 主图标 SVG 不匹配
- **原型**：使用对话气泡 SVG 图标
- **当前**：使用相同的对话气泡 SVG 图标（这个是正确的）

## 修复计划

### 步骤 1：更新标题和描述内容
- 将标题保持为"欢迎使用 AutoMate"
- 将描述改为"选择左侧的智能体开始对话，体验智能化的任务处理"

### 步骤 2：修正统计卡片样式
- 将卡片背景改为 `bg-gradient-to-br from-gray-100 to-gray-200`
- 将统计数字改为纯色（蓝色、绿色、紫色），不使用渐变
- 将统计标签字体大小改为 `text-xs`，添加 `mt-1` 间距

### 步骤 3：重构快速开始引导
- 移除当前的4个步骤结构
- 改为3个步骤结构，使用原型中的 `quick-start-guide` 和 `quick-start-item` 类
- 更新步骤内容为：
  1. 从左侧列表选择一个智能体
  2. 在输入框中输入您的问题
  3. 按 Enter 发送消息开始对话
- 添加步骤编号样式（圆形背景，渐变色）

### 步骤 4：验证深色主题适配
- 确保所有样式在深色主题下正确显示
- 参考原型中的深色主题样式

### 步骤 5：测试验证
- 在浏览器中打开 http://localhost:3000/
- 对比原型设计，确保完全一致
- 测试浅色和深色主题切换

## 验收标准

1. 标题显示"欢迎使用 AutoMate"
2. 描述显示"选择左侧的智能体开始对话，体验智能化的任务处理"
3. 统计卡片使用 `bg-gradient-to-br from-gray-100 to-gray-200` 背景
4. 统计数字使用纯色（蓝色、绿色、紫色）
5. 快速开始引导包含3个步骤，内容与原型完全一致
6. 所有样式在浅色和深色主题下正确显示
7. UI 还原度 ≥ 95%
