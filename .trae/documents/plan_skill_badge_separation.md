# 计划：分离技能徽章与消息内容

## 目标
将已激活的技能徽章从消息气泡内部移动到消息气泡下方，独立展示。

## 当前布局
```
┌─────────────────────────────┐
│ 技能徽章（紫色）             │  ← 在消息气泡内
├─────────────────────────────┤
│ 消息内容                    │
│ 【关于更新政务手机管理版...】│
└─────────────────────────────┘
```

## 目标布局
```
┌─────────────────────────────┐
│ 消息内容                    │
│ 【关于更新政务手机管理版...】│
└─────────────────────────────┘
┌─────────────────────────────┐
│ 技能: official-doc-optimize │  ← 移到消息气泡下方
│ 已激活                      │
└─────────────────────────────┘
```

## 修改步骤

### 1. 修改 EnhancedMessageBubble.tsx

将 `renderSkillActivation()` 调用从消息气泡内部移到大容器外部，使其成为独立的元素：

- 第146-167行：修改组件的返回结构
- 将技能徽章从 `div.max-w-[800px]` 内部移到外部
- 技能徽章使用独立的容器，与消息内容分离
- 保持原有的样式和功能不变

### 2. 验证
- 检查消息气泡与技能徽章之间的间距
- 确保暗色/亮色主题都正常显示
- 验证技能徽章的样式与原设计一致
